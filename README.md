Muttville AI Operations Prototype

This project explores how AI can help nonprofit animal shelters reduce operational overhead by consolidating information spread across Google Sheets, Slack, and Shelterluv.

Current capabilities
- Reads foster information from Google Sheets
- Exposes FastAPI endpoints
- Returns consolidated dog information

Future work
- Slack integration
- Shelterluv integration
- AI-powered search across operational notes

Status
Early prototype built in collaboration with Muttville Senior Dog Rescue.
